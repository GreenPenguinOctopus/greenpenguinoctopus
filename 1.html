Data Structure I


Course Description

The purpose of this course is to introduce the exciting world of data structures and an issue central to the art of computer programming.At the end of the course you will be equipped with the tools of data organization to enable you to write simple, clear, and efficient programs.

Our programming language of this course is Python.Hence, it is required that you have some programming experience in Python. To understand the core concepts of Data Structure is very helpful to solve large and complex problems and through this course you can learn a lot of new approaches of program design.
